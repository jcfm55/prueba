QUE GIT

Es un sofware de control de versiones que registra los cambios
realizados sobre un archivo o conjunto de archivos a lo largo
del tiempo

GIT nos permite gestionar distintas versiones de un mismo archivo
pudiendo volver a una version anterior o una más reciente 
cuando sea necesario

QUE ES UN REPOSITORIO
=====================
- Cuando trabajamos con GIT lo que creamos es un REPOSITORIO
  es decir, un lugar donde guardamos y administramos nuestros
  archivos

- Existe dos tipos de repositorios:
    - LOCALES Son aquellos que se crean de forma "local" en 
      nuestro PC, y que no necesariamente son compartidos.
    - REMOTOS Son aquellosque se encuentran alojados en algun
      servidor externo y que puede ser accedido desde 
      culaquier lugar. Un ejemplo de repositorio remoto puede
      ser Git-HUB.
